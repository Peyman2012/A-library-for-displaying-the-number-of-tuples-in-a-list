# A-library-for-displaying-the-number-of-tuples-in-a-list
A library for displaying the number of tuples in a list

#### Example :

    obj = num_tuple([8, 4, (4, 4, 5)])
    obj2 = num_tuple([8, 4, (4, 4, 5),2,4,(1,4,5)])
